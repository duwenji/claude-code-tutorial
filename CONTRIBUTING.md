# Contributing

このチュートリアルへの貢献を歓迎します。

## 貢献の方法

1. Issue で改善提案・誤りの報告
2. Pull Request でコンテンツの追加・修正
3. サンプルスキルの追加

## ドキュメント規約

- 日本語で記述
- 各セクションの冒頭に「学習時間」と「難易度」を記載
- コードブロックには言語指定を付ける
- セクション末尾に「次のステップ」へのリンクを付ける

## ファイル命名規則

- `docs/<part>/<number>-<kebab-case>.md`
- 例: `docs/01-core-features/02-slash-commands.md`
