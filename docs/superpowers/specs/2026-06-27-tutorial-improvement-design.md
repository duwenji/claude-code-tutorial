## 目的

Claude Code チュートリアル（日本語、7パート27セクション）を以下の3軸で改善する。

1. **内容の正確性・最新化** — 公式ドキュメント（docs.anthropic.com）を参照し、誤情報・古い情報を修正する
2. **実践演習の追加** — 各セクション末尾に練習問題（3〜5問）を追加し、学習効果を高める
3. **カバレッジの拡充** — 公式ドキュメント調査で発見した未収録の重要機能を加筆する

---

## 進め方: パート単位スプリント

Part 0 から Part 6 まで、パートごとにスプリントを実施する。

### 各スプリントの手順

```
Step 1: 公式ドキュメント調査
        そのパートのテーマに関する公式ページを Web 検索・取得

Step 2: 差分の洗い出し
        現チュートリアルと公式ドキュメントを比較し
        ・誤情報・古い情報
        ・未収録の重要機能
        をリストアップ

Step 3: セクション更新
        各セクションを修正・加筆（正確性 + カバレッジ）

Step 4: 演習追加
        各セクション末尾に「練習問題」ブロックを追加（3〜5問）
```

---

## 演習フォーマット

各セクションの末尾（「次のステップ」リンクの直前）に以下の形式で追加する。

```markdown
## 🏋️ 練習問題

1. **【確認】** ～を実行すると何が起きますか？
2. **【実践】** ～を設定してみましょう。
3. **【応用】** ～の場合、どのように対処しますか？
```

### 問題タイプ

| タイプ | 目的 | 例 |
|--------|------|----|
| 【確認】 | 知識の定着確認 | 「`claude doctor` は何をするコマンドですか？」 |
| 【実践】 | 手を動かす体験 | 「プロジェクトディレクトリで `claude` を起動してみましょう」 |
| 【応用】 | 考える力を鍛える | 「チームで CLAUDE.md を共有する場合の注意点は？」 |

各セクションに3〜5問（確認1〜2問 + 実践1〜2問 + 応用0〜1問）を目安とする。
